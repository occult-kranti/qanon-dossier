## Role Descriptions

### Project Coordinator (T001) - Lead
**Status**: Master Planner
- Oversee all operations
- Create and manage project roadmaps
- Schedule experiments and deliverables
- Track progress metrics
- Remove blockers
- Make final decisions

### Data Engineer (T002)
**Status**: Data Specialist
- Manage all data sources
- Create data pipelines
- Validate data quality
- Version data assets
- Document schemas

### Experiment Executor (T003) - Lead
**Status**: Execution Specialist
- Run scheduled experiments
- Log results and metrics
- Report issues
- Ensure reproducibility
- Manage experiment queue

### ML Engineer (T004)
**Status**: Model Specialist
- Develop clustering models
- Implement topic models
- Tune hyperparameters
- Create evaluation metrics
- Share findings

### Data Analyst (T005)
**Status**: Analysis Specialist
- Analyze experiment results
- Create visualizations
- Identify patterns
- Generate insights
- Create reports

### Web Developer (T006)
**Status**: Frontend Specialist
- Build interactive dashboards
- Design user interfaces
- Integrate visualizations
- Optimize performance
- Deploy to GitHub Pages

### DevOps Engineer (T007) - Lead
**Status**: Infrastructure Specialist
- Set up CI/CD pipelines
- Manage dependencies
- Automate deployments
- Monitor systems
- Handle versioning

### Technical Writer (T008)
**Status**: Documentation Specialist
- Write methodology docs
- Create guides
- Document code
- Maintain README
- Write reports

### QA Engineer (T009)
**Status**: Quality Specialist
- Validate outputs
- Create test plans
- Identify issues
- Verify reproducibility
- Report findings

### Research Lead (T010) - Expert
**Status**: Innovation Specialist
- Research new methods
- Prototype techniques
- Benchmark algorithms
- Propose improvements
- Lead innovation

---

**Notes**:
- (L) indicates team leads
- All team members report to Project Coordinator
- Cross-functional collaboration encouraged
- Regular sync meetings: Daily standups, Weekly reviews
